# DevLink — Developer Portfolio Builder

> **Build a developer brand that feels alive.**
> Clean, sharp, and ready for launch.

---

## 📌 Overview

**DevLink** is a developer-focused portfolio builder that lets developers create a personalized, live portfolio page — shareable via a custom URL. It also includes a discovery feed for finding other developers and a collaborative idea board called **SparkForge**.

Whether you're looking for freelance opportunities, building in public, or finding collaborators for your next project, DevLink brings everything into one place.

---

## ✨ Features

### 🧑‍💻 Portfolio Builder
- Create a public-facing developer profile with a **custom URL** (e.g., `/yourname`)
- Showcase your **skills, bio, and availability** status
- Multiple **portfolio themes** (e.g., `editorial`, `bento`) and **color schemes** (e.g., `modern`, `lavender`)
- **QR code generation** for instant mobile sharing
- One-click sharing to LinkedIn, X (Twitter), WhatsApp, and Email

### 🔍 Discover
- Browse published developer portfolios across the platform
- **Search** by name, username, bio, or skills
- Explore developer profiles with their tech stack and theme preferences

### ⚡ SparkForge
- Post your **project ideas** and find developers to build with
- Discover what others are building — search by concept, stack, or tag
- Send **"sparks"** to connect, then move to chat or direct contact

### 📥 Inbox
- Run **outreach and replies** in one centralized inbox
- Track new inquiries, pulse messages, and connection requests
- Live stats: profile views, new inquiries, unread pulses

### 🌙 Theme Toggle
- Built-in **dark/light mode** toggle across all pages
